# Creator

Внутренний сервис для генерации макетов по готовым шаблонам. Используется для быстрого создания изображений и текстовых файлов под разные форматы: веб, посевы, SMM и другие рабочие сценарии.

## Что умеет сервис

- выбор нужного макета из интерфейса;
- заполнение полей: город, адрес, дата, телефон, ссылка и другие данные;
- генерация изображений по шаблонам;
- экспорт PNG/SVG и текстовых файлов;
- сборка результата в ZIP;
- авторизация через локальный `auth.json`;
- SSO-вход через внешний корпоративный сервис при наличии общего секрета.

## Стек

- PHP;
- HTML/CSS/JavaScript;
- GD Library;
- Python;
- fontTools для перевода текста в SVG-кривые;
- Morphos для работы с русскими склонениями.

## Подготовка к запуску

1. Скопировать пример конфига:

```bash
cp config.local.example.php config.local.php
```

2. Заполнить настройки в `config.local.php`.

3. Создать файл авторизации:

```bash
cp json/auth.example.json json/auth.json
```

4. Указать свой логин и пароль в `json/auth.json`.

5. Установить Python-зависимость для SVG-экспорта:

```bash
pip install fonttools
```

6. Проверить, что PHP работает с расширением GD.

7. Добавить необходимые шрифты в папку `fonts/`.

## Важный момент про шрифты

Шрифтовые файлы не добавлены в GitHub-версию. Их нужно хранить отдельно и добавлять на сервер вручную. Код ожидает шрифты в папке `fonts/`, имена файлов указаны в настройках макетов.

## Важные файлы и папки

- `index.php` — основной интерфейс;
- `auth.php` — авторизация;
- `create.php` — генерация макетов и ZIP;
- `functions.php` — основная логика генерации;
- `makets.php` и `configs/` — настройки макетов;
- `templates/` — исходные изображения-шаблоны;
- `text_to_path.py` — перевод текста в SVG-кривые;
- `morphos/` — библиотека для работы с русскими словами.
